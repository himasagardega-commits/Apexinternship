# Data Dictionary

| Column Name | Description | Data Type |
|-------------|-------------|-----------|
| Order_ID | Unique order identifier | String |
| Order_Date | Date of purchase | Date |
| Customer_ID | Customer identifier | String |
| Customer_Name | Name of customer | String |
| Age | Customer age | Numeric |
| Gender | Gender of customer | String |
| City | Customer city | String |
| Product | Product purchased | String |
| Category | Product category | String |
| Quantity | Number of items purchased | Numeric |
| Unit_Price | Price per unit | Numeric |
| Total_Sales | Total revenue generated | Numeric |
